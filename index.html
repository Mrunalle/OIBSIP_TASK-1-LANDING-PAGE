Web Design using
HTML & CSS
Course

HTML (Hypertext Markup Language) is the standard markup language which creates web pages.
CSS (Cascading Style Sheets) describes how HTML elements should look.
HTML tells web browser (like Chrome or Internet Explorer) what text and images it needs to display.
HTML also tells where these text and images should appear.
CSS describes how these images or text should look like.
By adding some simple CSS styling to your webpage, you can make it look more interesting!
Now that you know the use of HTML and CSS, lets start building our webpage!
